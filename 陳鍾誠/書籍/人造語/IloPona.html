IloPona -- English Example (人造語英文版範例)

《人造語》是衍生自《道本語》的一種語言。道本語成功的使用僅僅 120 個單字就能講出幾乎所有的事情，你可以閱讀下列範例之後，看看自己是否能理解這些語句！

說明：符號 ' 念成 li， / 念成 e 。

IloPona is derived from TokiPona with (Chinese/English) grammar. TokiPona has proved one theory : We can speak in only 120 words. You may read the following Ilo Pona Examples to test if you can understand it or not.

Comment : The symbol ' is pronounced as li, / is pronounced as e.
